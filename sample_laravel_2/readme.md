This is example of laravel CMS that allows user to create and support some eevents (festivals, summits, etc.). It provides 3 main blocks for site control:
1) Content manager that allows to craete and edit site pages with convenient composer with many templates. This block also allows to create image galleries and upload other media content.
2) Delegate manager that provides instruments for registration and control events visitors, and mail sending to visitors.
3) Meeting manager that allows to plan events grid inside your main event, craete group of visitors, control visitors timeslots, etc.
